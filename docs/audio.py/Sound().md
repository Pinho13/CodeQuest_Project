# Sound()

```py
Sound(sound: str, volume: float = 1)
```

  * **sound:** String with the path of the sound
  * **volume:** Volume of the sound

## Properties

  * **sound:** Instance of the sound

## Callable Functions

  * **set_volume(volume: float):** Sets sound volume
  * **play():** Plays sound
  * **stop():** Stops sound
